MISE A JOUR PRE-PUBLICATION + SESSION

Nouveautés:
1) Le client reste connecté en naviguant entre les menus grâce au stockage localStorage.
2) Le menu affiche Dashboard client / Dashboard développeur si l'utilisateur est déjà connecté.
3) Un bouton Déconnexion supprime la session et renvoie vers la page login.
4) Dans le Dashboard client et le Lab Algorithmique, la mission est optimisée AVANT publication:
   - analyser mission
   - optimiser budget
   - améliorer description
   - générer cahier des charges
   - exporter PDF via impression navigateur
   - recommander au meilleur développeur
   - publier mission optimisée

Route principale:
http://localhost:4200/advanced-lab

Pour exporter PDF:
cliquer Exporter PDF, puis choisir "Enregistrer au format PDF" dans la fenêtre d'impression.
